AS AN END USER

As a user, I can create a new support ticket with a title and description so I can request help. (3)

As a user, I can choose a ticket type from a dropdown so my request is routed appropriately. (2)

As a user, I can attach one or more files to my ticket so I can provide context. (5)

As a user, I can add optional priority (e.g., Low/Medium/High) so staff can triage my ticket. (2)

As a user, I can see a confirmation with my ticket ID after submission so I know it was received. (1)

As a user, I can view a list of my submitted tickets so I can track status. (3)

As a user, I can open a ticket to see its details (status, assignee, comments, history). (3)

As a user, I can comment on my open ticket to add updates or answer staff questions. (3)

As a user, I can edit my ticket’s description before it is picked up by staff in case I need to clarify. (5)

As a user, I can close my own ticket if I no longer need help. (2)

As a user, I can login to my account securely with my google account. (1)
